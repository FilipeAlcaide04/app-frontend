## Quick Start (3 minutes per animation)

1. Go to https://www.mixamo.com/ (free with Adobe account)
2. Search for animation name (e.g., "Bored")
3. Click "Download"
4. Settings:
   - Format: **FBX for Unity (.fbx)**
   - Skin: **Without Skin**
   - Frame Rate: **30**
5. Rename to exact name above (e.g., `Bored.fbx`)
6. Save here in this folder
7. Restart dev server (`npm run dev`)
8. Done! 🎉

## Important Notes

- **No conversion needed** - FBX works directly!
- File names are case-sensitive
- Use hyphens, not spaces
- Extension must be `.fbx`
- ~100-500KB per file
- Application works without animations (they're optional)

See `COMO_ADICIONAR_ANIMACOES.md` in project root for detailed Portuguese instructions.
