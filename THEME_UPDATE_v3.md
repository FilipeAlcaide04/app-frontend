## 🎨 Nova Paleta de Cores

### Cores Base
```css
--background:    oklch(0.95 0.005 265)   /* Quase branco, tom azulado */
--foreground:    oklch(0.18 0.008 260)   /* Texto: cinzento escuro */
--card:          oklch(0.98 0.003 265)   /* Branco puro */
--border:        oklch(0.88 0.008 268)   /* Cinza muito claro */
```

### Cores de Interação
```css
--primary:       oklch(0.50 0.22 265)    /* Azul GitHub profissional */
--accent:        oklch(0.58 0.28 305)    /* Roxo vibrante */
--secondary:     oklch(0.85 0.010 270)   /* Cinzento suave */
--muted:         oklch(0.80 0.008 268)   /* Texto atenuado */
```

### Cores de Status
```css
--status-active:   oklch(0.58 0.28 305)  /* Roxo */
--status-online:   oklch(0.55 0.20 144)  /* Verde */
--status-error:    oklch(0.60 0.22 27)   /* Vermelho */
```
